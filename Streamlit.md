# Streamlit

### Prepare
- Python 3.9 ~ 3.11
- PIP
- github push authority

- package 관리 도구에 관계없이 가상환경에서 실행할 것
  - venv, pipenv, poetry, virtualenv, conda
    - (venv 환경에서 진행함)

```shell
pip3 install --upgrade pip

pip install streamlit

streamlit hello

```
